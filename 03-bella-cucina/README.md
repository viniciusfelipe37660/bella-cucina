# Bella Cucina — Landing Page de Restaurante Italiano

Landing page elegante para restaurante, desenvolvida por **Vinicius Felipe** como projeto de portfólio. Destaque para o sistema de cardápio com filtros e SEO avançado com Structured Data.

## 🔗 Demo
> Após o deploy: `https://seu-usuario.github.io/bella-cucina`

## Sobre o Projeto
Landing page completa para um restaurante italiano de alto padrão, com sistema de cardápio filtrado por categoria, formulário de reserva de mesa e galeria de fotos com efeitos hover.

## Tecnologias
- HTML5 Semântico
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript Vanilla (ES6+)
- JSON-LD (Structured Data para SEO)

## Funcionalidades
- Navbar transparente que muda de cor ao fazer scroll
- Menu mobile responsivo com animação hambúrguer → X
- Hero com imagem de fundo e copy persuasivo
- Barra de diferenciais com ícones
- **Cardápio filtrado por categoria** (Massas / Pizzas / Entradas / Sobremesas) com animação de entrada
- Seção "Nossa História" com quote e fotos sobrepostas
- **Galeria de fotos** com efeito hover e legenda deslizante
- Formulário de reserva com validação de dia de folga (segunda-feira)
- Depoimentos de clientes
- Botão flutuante do WhatsApp
- **JSON-LD (Schema.org)** para SEO de restaurante no Google
- SEO completo: meta description, Open Graph, Twitter Card, canonical URL
- Máscara de telefone em JavaScript

## Deploy no GitHub Pages

### Passo a passo:
1. Crie um repositório chamado `bella-cucina`
2. Na pasta do projeto:
   ```bash
   git init
   git add .
   git commit -m "feat: landing page Bella Cucina"
   git branch -M main
   git remote add origin https://github.com/SEU-USUARIO/bella-cucina.git
   git push -u origin main
   ```
3. No GitHub: **Settings → Pages → Source: main / root → Save**
